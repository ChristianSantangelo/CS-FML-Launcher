1). Allocate More RAM

When playing with a lot of mods, it is important to make sure that you have allocated more of your systems resources to help run and maintain a smooth FPS. Depending on the mods you are running, you should use 4-6GB of RAM allocated to Minecraft. Learn more about using JVM Arguments to increase your RAM Allocation, here.

2). Use Performance Mods

Not only is RAM important to your gameplay, it is also important to use as less RAM as possible, while still running the game smoothly. This is important so if you were running big mods, you would still have room for other mods, and room for your computer to work on other programs in the background. Popular performance mods include Optifine, BetterFPS, and FoamFix for Forge. For Fabric, you would use Lithium, Sodium, Phospher, and Starlight. Keep in mind, that you will most likely get more FPS using Fabric performance mods.

3). Organize Your Mods

If you are an avid modded player, you know the struggle of finding mod, which is why we try to put them in a place where we can find them. I personally have a 'mods' folder on my desktop for each modpack I have. However, if you use Forge, you can place folders within your mods folder, which can organize your mods for you. Keep in mind that this will only apply for versions. If you want to read more about this, click here.
